# Dark New Tab 2
A minimalistic dark new tab page for Google Chrome. With bookmarks from "bookmarks bar" folder.
No external network requests. No analytics. No tracking. No bullshit.

### What?
1. A minimalistic dark new tab page replacement for Chrome.
1. New tab page have bookmarks.

### Why?
1. I don't want to be blinded by a white page whenever I open a new tab.
1. I need bookmarks on new tab only.
